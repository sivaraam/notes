# Database Engineering - Mastery

The goal is to get a good grasp of the following databases:

1. [PostgreSQL](./postgres) - RDMS
2. MongoDB - document database
3. Redis - in-memory key-value DB
4. InfluxDB - Time series DB
5. neo4J - Graph DB
